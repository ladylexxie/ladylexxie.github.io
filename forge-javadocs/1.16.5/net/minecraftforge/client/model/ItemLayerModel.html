Class ItemLayerModel

  • java.lang.Object
    • net.minecraftforge.client.model.ItemLayerModel
  • All Implemented Interfaces:
    IModelGeometry<ItemLayerModel>
    public final class ItemLayerModel
extends java.lang.Object
implements IModelGeometry<ItemLayerModel>
    Forge reimplementation of vanilla ItemModelGenerator, i.e. builtin/generated models, with the following changes: - Represented as a true IUnbakedModel so it can be baked as usual instead of using special-case logic like vanilla does. - Various fixes in the baking logic. - Not limited to 4 layers maximum.
