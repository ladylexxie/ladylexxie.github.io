Class AddAttributeTooltipsEvent

java.lang.Object
net.neoforged.bus.api.Event
net.neoforged.neoforge.client.event.AddAttributeTooltipsEvent
public class AddAttributeTooltipsEvent extends net.neoforged.bus.api.Event
This event is fired after attribute tooltip lines have been added to an item stack's tooltip in AttributeUtil.addAttributeTooltips(net.minecraft.world.item.ItemStack, java.util.function.Consumer<net.minecraft.network.chat.Component>, net.neoforged.neoforge.common.util.AttributeTooltipContext).

It can be used to add additional tooltip lines adjacent to the attribute lines without having to manually locate the inject point.

This event may be fired on both the logical client and logical server.
