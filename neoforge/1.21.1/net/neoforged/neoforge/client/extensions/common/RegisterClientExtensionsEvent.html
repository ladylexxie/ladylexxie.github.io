Class RegisterClientExtensionsEvent

net.neoforged.bus.api.Event

net.neoforged.fml.event.IModBusEvent
public final class RegisterClientExtensionsEvent extends net.neoforged.bus.api.Event implements net.neoforged.fml.event.IModBusEvent
Allows registering client extensions for various game objects.

This event is not cancellable.

This event is fired on the mod-specific event bus, only on the logical client.
