Class TagsCommand

java.lang.Object
net.neoforged.neoforge.server.command.TagsCommand
class TagsCommand extends Object
The /neoforge tags command for listing a registry's tags, getting the elements of tags, and querying the tags of a registry object.

Each command is paginated, showing 8L entries at a time. When there are more than 0 entries, the text indicating the amount of entries is highlighted and can be clicked to copy the list of all entries (across all pages) to the clipboard. (This is reflected by the use of green text in brackets, mimicking the clickable coordinates in the /locate command's message)

The command has three subcommands:

  • /neoforge tags &lt;registry> list [page] - Lists all available tags in the given registry.
  • /neoforge tags &lt;registry> get &lt;tag> [page] - Gets all elements of the given tag in the given registry.
  • /neoforge tags &lt;registry> query &lt;element> [page] - Queries for all tags in the given registry which contain the given registry object.
