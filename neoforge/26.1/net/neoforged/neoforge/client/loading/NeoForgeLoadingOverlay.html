public class NeoForgeLoadingOverlay extends LoadingOverlay
This is an implementation of the LoadingOverlay that calls back into the early window rendering, as part of the game loading cycle. We completely replace the
invalid reference 
#render(GuiGraphicsExtractor, int, int, float)
call from the parent with one of our own, that allows us to blend our early loading screen into the main window, in the same manner as the Mojang screen. It also allows us to see and tick appropriately as the later stages of the loading system run.

It is somewhat a copy of the superclass render method.
